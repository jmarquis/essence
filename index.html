Minimalist

No dependencies. No frameworks. No clunky CSS classes. In fact, no CSS output at all until you actually use the grid mixins. It’s just a single .less or .sass file with a simple set of mixins.

Responsive

Completely fluid, completely flexible. Use different numbers and sizes of cells in each container. Define custom gutters or breakpoints. Use media queries to adjust layouts. It’s responsive layouts done right.

Semantic

No layout-related CSS classes. Keep your styles in your stylesheets. Apply layouts styles using single-line LESS mixins and keep your markup squeaky clean.

Demo

Resize your browser to see how this grid rearranges itself to the available space using media queries and single-line mixins.
