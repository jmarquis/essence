{% highlight less %}@import "essence";

.content {
    #grid > .row();

    article {
        #grid > .cell( (8/12) );
    }

    aside {
        #grid > .cell( (3/12) );
        #grid > .offset( (1/12) );
    }
}{% endhighlight %}

Minimalist

No dependencies. No frameworks. No clunky CSS classes. In fact, no CSS output at all until you actually use the grid mixins. It’s just a single .less file with a single #grid namespace.

Responsive

Completely fluid, completely flexible. Use any container of any size as a grid row. Use different numbers and sizes of cells in each row, or define custom gutters or breakpoints. Use media queries to adjust layouts. It’s responsive grids done right.

Semantic

No layout-related CSS classes. Keep your styles in your stylesheets. Apply grid layouts using single-line LESS mixins and keep your markup squeaky clean.
